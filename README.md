## Як передати Claude Code

1. Розпакуй цей архів у свою робочу папку
2. Відкрий Claude Code у терміналі в цій папці:
   ```
   cd recreate-media
   claude
   ```
3. Відправ промпт:
   ```
   Ось промпт і матеріали для нового проєкту.
   Промпт у PROMPT_FOR_CLAUDE_CODE.md, всі фото і прототип — в цій же папці. Виконай.
   ```
4. Claude Code прочитає промпт, спитає уточнення (якщо потрібно), і згенерує повноцінний Next.js проєкт у підпапці.

Час на генерацію: 10–20 хвилин.
